[0:00 — HOOK]
Hello everyone, and welcome to this demonstration of the Expense Auditor. In today's corporate world, manually auditing employee travel and entertainment expenses is a slow, error-prone process. It is vulnerable to non-compliant spending, duplicate submissions, and security risks like prompt injections or sensitive data leakage. The Expense Auditor solves this by automating policy checks, filtering out security threats, and providing a seamless human-in-the-loop review interface.

[0:20 — WHAT IT IS]
Built for the "Agents for Business" track using the Google Agent Development Kit 2.0, this project coordinates multiple specialized AI agents, checks claims against live databases using a Model Context Protocol server, and automatically redacts PII before processing.

[0:40 — SHOW THE BANNER]
(show assets/cover_page_banner.png on screen)
As you can see from our cover banner, the Expense Auditor focuses on three pillars: Multi-Agent coordination, Security Checkpoints, and Model Context Protocol database integrations. Let's walk through the architecture to see how these fit together.

[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png on screen)
Looking at the workflow diagram, every expense submission begins at the Security Checkpoint node. This node filters out prompt injection attacks, redacts SSNs or credit card numbers, and screens for restricted expense items. Safe inputs then reach our Orchestrator Coordinator Agent. 
(pause and point to Orchestrator)
The Orchestrator acts as the central router and delegates tasks. It calls the Policy Auditor Agent, which queries lodging and food caps via our local Model Context Protocol server, and checks for double-submissions against historical database records. 
(pause and point to Human node)
If a claim is clean but exceeds $1,000, the workflow pauses at the Human Review node. Here, the system suspends execution and requests input from an administrator in the playground UI. Once a decision is made, the workflow resumes and routes to the Communication Drafter Agent to compile the final approval or rejection email.
