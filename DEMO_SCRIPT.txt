[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Looking at our system architecture, the workflow begins at the Security Checkpoint node which filters prompt injections and redacts sensitive PII. Safe inputs proceed to the Orchestrator Coordinator Agent. The Orchestrator delegates deep checks to the Policy Auditor Agent, which accesses corporate lodging limits and logs transactions using a local Model Context Protocol server. If a claim exceeds threshold limits, the workflow suspends at the human review pause node until an admin responds, before routing to the Communication Drafter Agent to create the final email response.
